# CursorMonitor
CUI of Mouse cursor Position and RGB monitoring

マウスカーソルの位置とその点の色(RGB値)を可視化するCUIのPythonコードです。
実行には、Pyautoguiが必要です。

Pyautoguiが未インストールの場合は、下記コマンドにてインストールを行ってください。
```
$pip3 install pyautogui
```


### CursorMonitorの実行
```
$python3 cursor_monitor.py
```

### CursorMonitorの終了
コンソールを閉じて終了
